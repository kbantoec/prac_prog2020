Tutoriel 2

Ici, nous allons revoir comment construire la classe Fraction de A à Z, pas à pas.

class Fraction: pass if __name__ == '__main__': # bloc main pass
Just press 'Run'.

Premièrement, par convention, on écrit le nom d'une classe avec une majuscule au début. Donc, voilà pourquoi on écrit class Fraction et non class fraction!

Tu remarqueras que class Fraction(): est équivalent à class Fraction:. Le dernier cas est préconisé par PEP 8 (le guide de style de Python) dans le cas où il n'y a pas d'héritage (notion qu'on verra à la semaine 5).

La commande de pass est une opération nulle; rien ne se passe quand elle s'exécute. pass est également utile dans les endroits où ton code sera éventuellement utilisé, mais n'a pas encore été écrit.

J'ai pu observer que certains étudiants ne comprenaient pas ce qu'est le bloc main. Tout d'abord, par "bloc main" on entend if __name__ == '__main__':. Et ce qui est important de retenir, c'est que tout ce que tu vas écrire après cette ligne est dans le but de tester ton module.

Donc, dorénavant, à chaque fois que le mot "tester" vient à toi pour un morceau de code, tu le placeras dans le bloc main!
